🔴 Supabase Setup Required

WhyRobux menggunakan Supabase — database PostgreSQL gratis & open-source yang populer di kalangan developer. Ikuti langkah berikut untuk mengaktifkan autentikasi nyata:

Cara Setup (5 menit):

1. Buka supabase.com → Create account (gratis)
2. Buat project baru → catat Project URL & anon key
3. Di Supabase Dashboard → Authentication → Providers → aktifkan Email
4. Jalankan SQL ini di SQL Editor:

create table profiles ( id uuid references auth.users primary key, username text unique not null, role text default 'buyer', balance numeric default 0, roblox_username text, created_at timestamptz default now() );
5. Masukkan kredensial di bawah ini:

Kredensial disimpan di localStorage browser Anda. Tidak dikirim ke mana pun selain Supabase.
